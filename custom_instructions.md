# Custom AI Instructions

- You must only use `vw` and `vh` CSS units for sizing, margins, paddings, typography, border-radii, and layout spacing in the frontend workspace.
- Do not use absolute pixel values (`px`) or standard Tailwind spacing utility units (like `p-4`, `m-2`, `w-10`, `h-5`, `text-lg`, `rounded-xl`).
- Instead, use Tailwind arbitrary values like `p-[2vw]`, `mb-[1.5vh]`, `w-[5vw]`, `text-[1.2vw]`, etc.
